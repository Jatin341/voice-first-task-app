# Voice First Task App

A full-stack voice-enabled task management application built using Spring Boot and React.

## Features
- Voice task input
- Add tasks
- JWT Authentication
- Task analytics
- React frontend
- Spring Boot backend

## Tech Stack
- React
- Spring Boot
- MySQL
- JWT
- Web Speech API

## Run Frontend
cd frontend
npm install
npm run dev

## Run Backend
cd voicetask
mvn spring-boot:run
